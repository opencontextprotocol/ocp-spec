The Open Context Protocol (OCP) provides multiple resources for documentation and implementation:

- **User Documentation**: Visit [opencontextprotocol.io](https://opencontextprotocol.io) for comprehensive user-facing documentation.
- **Go SDK**: The official Go implementation is available at [github.com/opencontextprotocol/go-sdk](https://github.com/opencontextprotocol/go-sdk) - [Issues](https://github.com/opencontextprotocol/go-sdk/issues)
- **Specification**: The core OCP specification is available at [github.com/opencontextprotocol/specification](https://github.com/opencontextprotocol/specification) - [Discussions](https://github.com/opencontextprotocol/specification/discussions)
- **TypeScript SDK**: The TypeScript implementation can be found at [github.com/opencontextprotocol/typescript-sdk](https://github.com/opencontextprotocol/typescript-sdk) - [Issues](https://github.com/opencontextprotocol/typescript-sdk/issues)

For questions or discussions, please open a discussion in the appropriate GitHub repository based on your implementation or use case. You can also visit the [Open Context Protocol organization on GitHub](https://github.com/opencontextprotocol) to see all repositories and ongoing development.
