This page lists the different revisions of the Open Context Protocol.  
See [Versioning]({{< ref "/specification/draft/basic/versioning" >}})  
for details on how protocol versions are managed.

Revisions may be marked as:

- Draft: in-progress specifications, not yet ready for adoption.
- Current: the active protocol version, which may receive [backwards compatible updates]({{< ref "/specification/draft/basic/versioning" >}}).
- Final: past, stable specifications that will not change.
